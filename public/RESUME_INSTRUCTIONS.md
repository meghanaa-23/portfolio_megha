# How to Add Your Resume

## Step 1: Convert DOCX to PDF

You currently have `resume.docx` in the public folder. You need to convert it to PDF:

### Option A: Using Microsoft Word
1. Open `resume.docx` in Microsoft Word
2. Click File → Save As
3. Choose "PDF" as the file type
4. Save as `resume.pdf` in the `public` folder

### Option B: Using Google Docs
1. Upload `resume.docx` to Google Drive
2. Open it with Google Docs
3. Click File → Download → PDF Document (.pdf)
4. Save as `resume.pdf` in the `public` folder

### Option C: Online Converter
1. Go to https://www.ilovepdf.com/word_to_pdf
2. Upload your `resume.docx`
3. Download the converted PDF
4. Save as `resume.pdf` in the `public` folder

## Step 2: Place the PDF

Once converted, place `resume.pdf` in the `public` folder (same location as this file).

## What You'll Get

✅ Resume viewer section on your portfolio
✅ View Resume button (opens in new tab)
✅ Download PDF button
✅ Embedded PDF viewer showing your resume
✅ Resume link in navigation menu

The resume will be visible at the bottom of your portfolio with view and download options!
